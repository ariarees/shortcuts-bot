# Clove Shortcuts

<div align="center">
This is a Discord bot created in TypeScript, using DiscordJS and Axios, with middleware to prevent unauthorized use.

</div>

## Features:
<div align="center">
The bot has a varity of functions, checking if Hytale usernames are free, explaining what items are, with links to videos and tutorials.

The bot also connects to the Doughmination SystemÂ® API, to allow Clove to switch fronts using a bot API token.
</div>
